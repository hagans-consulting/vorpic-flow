# Vorpic Flow
**Knowledge Base & SOP Generator · Part of the Vorpic Operations Suite by Hagans Consulting**

## Overview
Vorpic Flow is a dual-function operations tool for creating and storing standard operating procedures. Paste rough notes or upload source documents and the built-in Claude AI engine formats them into clean, structured SOP documents — complete with Purpose, Scope, Responsibilities, Procedure, and Notes sections. Generated SOPs are stored locally, searchable, and exportable.

Built for small and mid-size businesses, consultants, and operations teams who need to document processes fast without a complicated toolchain.

## Features
- AI-powered SOP generation via Claude (Anthropic API)
- Raw input support — paste notes, bullet points, or rough drafts
- File upload support for source documents (.txt, .md)
- Category tagging — General, HR, IT, Operations, Safety, Finance
- Editable output — refine AI-generated SOPs before saving
- Copy to clipboard and Print/Save as PDF export
- Document Library — searchable, filterable SOP storage
- IndexedDB storage with quota-exceeded handling
- Dark mode — full suite design system

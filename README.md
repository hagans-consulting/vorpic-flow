# Vorpic Flow
**Knowledge Base & SOP Generator · Part of the Vorpic Operations Suite by Hagans Consulting**

## Overview
Vorpic Flow is a dual-function operations tool for creating and storing standard operating procedures. Paste rough notes or upload source documents and the built-in AI engine formats them into clean, structured SOP documents — complete with Purpose, Scope, Responsibilities, Procedure, and Notes sections. Generated SOPs are stored in the cloud, searchable, and exportable.

Built for small and mid-size businesses, consultants, and operations teams who need to document processes fast without a complicated toolchain.

## What it does
- AI-powered SOP generation via Groq (Llama 3) — bring your own free API key
- Raw input support — paste notes, bullet points, or rough drafts
- File upload support for source documents (.txt, .md)
- Category tagging — General, HR, IT, Operations, Safety, Finance
- Editable output — refine AI-generated SOPs before saving
- Copy to clipboard and Print/Save as PDF export
- Document Library — searchable, filterable SOP storage with folders
- Cloud sync via Supabase — SOPs persist across devices and sessions
- Dark mode, Settings with HomeDeck navigation and sign out

## Setup
Flow requires a free Groq API key to generate SOPs. Get yours at [console.groq.com](https://console.groq.com) and paste it into Settings → AI Provider on first launch.
